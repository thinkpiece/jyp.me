Welcome to this website where I aim to provide some information on my past and present research work and publications. I'm trying to solve hard computational challenges for future technologies on computer vision applications.

Research Interest

My research interests mainly focus on energy-efficient SoC, mapping from software to hardware, especially for computer vision and machine learning algorithms. This includes:

  • Robust object recognition algorithms for real-world applications
  • Heterogeneous many-core architecture and parallel programming
  • Energy-efficient System-on-a-Chip (SoC) architecture for machine learning
